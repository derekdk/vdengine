# Audio Assets

This directory should contain audio files for the audio demo.

## Required Files

- **music.wav** (or .mp3, .ogg) - Background music track
- **beep.wav** (or .mp3) - Sound effect

## Where to Get Free Audio

### Music
- [FreePD](https://freepd.com/) - Public domain music
- [Incompetech](https://incompetech.com/music/) - Royalty-free music (requires attribution)
- [Purple Planet](https://www.purple-planet.com/) - Free music for projects

### Sound Effects
- [Freesound](https://freesound.org/) - Community uploaded sounds (various licenses)
- [Zapsplat](https://www.zapsplat.com/) - Free SFX library
- [Sonniss GDC Bundle](https://sonniss.com/gameaudiogdc) - Annual free sound packs

## Quick Start

1. Download a short music loop (WAV format recommended)
2. Rename it to `music.wav` or `music.mp3`
3. Download a short beep/click sound
4. Rename it to `beep.wav`
5. Place both files in this directory
6. Run the audio_demo example

## File Format Recommendations

- **Format**: WAV (most compatible), MP3, OGG also supported
- **Sample Rate**: 44100 Hz
- **Bit Depth**: 16-bit
- **Channels**: Stereo for music, mono for SFX
- **Music Length**: 10-60 seconds (will loop)
- **SFX Length**: < 2 seconds
